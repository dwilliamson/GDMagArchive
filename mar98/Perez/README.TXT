Portal Demo
by Adrian Perez
Game Developer Magazine

Included is all the sources, the executable, and the data file.

any questions can be directed to: amperez+@andrew.cmu.edu

To control the demo, move around using the arrow keys, use space bar
to stop movement, and use z and x to switch between overhead and camera
view.  the overhead view allows you to see the polygon set being clipped
and culled in realtime as polygons move into view.
