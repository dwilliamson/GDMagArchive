This ZIP file contains the sample source code that accompanies the article "Taming Vertex Data: Using C++ Templates" which appears in the October 2003 edition of Game Developer Magazine.

This ZIP file contains two installers.  

1) SourceCodeSetup.exe will install the sample source code which is unique to the magazine article.
2) Direct3DSamplesSetup.exe will install modified versions of the Direct3D samples.  These are subject to the Microsoft End-User License Agreement.

The modified D3D samples MUST be installed in your existing DirectX directory in order to compile.
